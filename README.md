## 🚀 Getting Started

1.  **Clone the Beast:**
    ```bash
    git clone https://github.com/Yudhajeet-debug/Quantum-Edge-AI.git
    ```
2.  **Install the Armoury (Using pnpm Power!):**
    ```bash
    pnpm install
    ```
3.  **Ignite the Engine:**
    ```bash
    pnpm run dev
    ```
